Placez ici le logo officiel utilise par l'ecran de connexion.

Fichiers reconnus automatiquement par `LoginController` :
- `logo.png`
- `logo-uadb.png`
- `logo-uadb.jpg`
- `logo-uadb.jpeg`

Chemin de chargement :
- `src/main/resources/assets/`

Conseils :
- preferez un fond transparent si possible
- une image carree d'au moins `512x512` donne un meilleur rendu
- le login utilisera cette image automatiquement si elle existe

Si aucun de ces fichiers n'est present, le logo dessine dans le code reste affiche en secours.
